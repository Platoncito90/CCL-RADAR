# CCL Radar v2

Monitor de brecha CCL en tiempo real para ADRs y CEDEARs argentinos.

## Features
- Calcula valor teórico ARS vs precio de mercado
- CCL implícito por instrumento
- Señales COMPRAR / VENDER / JUSTO
- Alertas por umbral de gap configurable
- Historial de snapshots
- Exportar a CSV
